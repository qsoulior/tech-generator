## 3. Выполнение программы

### 3.1. Запуск

Запуск выполняется командой:

```bash
./{{ .executable }} --config /etc/{{ .executable }}.conf
```

### 3.2. Основной режим

Оператор взаимодействует с программой через диалоговые формы. Каждая форма содержит подсказки и валидаторы ввода.

### 3.3. Завершение работы

Для штатного завершения используется команда меню «Выход» либо комбинация `Ctrl + Q`.

## 4. Сообщения оператору

| Код | Сообщение | Действие оператора |
| --- | --- | --- |
| 001 | Программа запущена | продолжить работу |
| 100 | Введены некорректные данные | проверить ввод и повторить |
| 200 | Связь с сервером недоступна | проверить сетевые настройки |
| 500 | Внутренняя ошибка | обратиться в службу поддержки |
